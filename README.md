# yuque-blog-back

语雀 blog

封装了 api<br />
[语雀的开发者文档](https://www.yuque.com/yuque/developer)

技术栈

-   nest.js

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## HMR

热更新，需要启动 2 个服务

```bash
$ npm run webpack

$ npm run start:hmr
```
